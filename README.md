# v2ray tcp+tls+web (beta)
automated script for v2Ray (TCP+TLS+Web)

# Usage
```
wget -N --no-check-certificate https://raw.githubusercontent.com/phlinhng/v2ray-tcp-tls-web/beta/v2.sh && chmod +x v2.sh && ./v2.sh
```
