# v2ray tcp+tls+web (beta)
automated script for v2Ray (TCP+TLS+Web)

## Usage
```
wget -O-  https://raw.githubusercontent.com/phlinhng/v2ray-tcp-tls-web/beta/v2.sh | bash -
```
